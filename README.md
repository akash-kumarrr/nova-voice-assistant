---

Nova is an AI-powered voice assistant — type a message, Nova thinks with **LangChain**, and speaks back using **pyttsx3**. No mic needed.

```
 ⌨️  You type  →  ⚡ FastAPI  →  🧠 LangChain  →  🔊 pyttsx3 speaks
```

---

<div align="center">

| Layer | Stack |
|:---:|:---|
| 🐍 Backend | Python · FastAPI |
| 🧠 AI | LangChain |
| 🔊 Voice | pyttsx3 |
| 📱 Frontend | Flutter · Dart |

</div>

---

```bash
# Backend
cd backend && pip install -r requirements.txt
uvicorn main:app --reload

# Frontend
cd frontend && flutter pub get && flutter run
```
